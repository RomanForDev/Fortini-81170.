# COMISIÓN 81170 - Coderhouse.

## Programación Backend I.

=============================================================

### How to install:

```bash

git clone https://github.com/RomanForDev/Fortini-81170.
cd Fortini-81170
npm install
npm i -g nodemon
npm i express

```

### On "package.json" add :

```bash
"type": "module",
```

#### Resta:

- Métodos POST, PUT y DELETE
- Path al carrito con:
  - GET y POST para añadir al carro y mostrar la información.

##### La carpeta node_modules está en .gitignore
